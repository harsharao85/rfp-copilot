## Question

Which SSO protocols are supported?

## Answer

We support SAML 2.0 and OIDC for customer-initiated SSO. SCIM 2.0 is supported for automated user provisioning. OAuth 2.0 Authorization Code + PKCE is used for native integrations. [sso_mfa_guide]
